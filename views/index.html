Charmed Image Search Abstraction Layer


This site uses a Google Search API to show image metadata based on user searches.

Enter your search term after https://charmed-isal.herokuapp.com/search/ in the URL bar.


Paginate through results by adding query?offset=[page number] to the end of the URL.


For example, search for Dolphins image metadata at:

https://charmed-isal.herokuapp.com/search/Dolphins


See Page 2 of Dolphins results at:

https://charmed-isal.herokuapp.com/search/Dolphins?offset=2


You may also see the 10 most recent searches by visiting:

https://charmed-isal.herokuapp.com/recent/
